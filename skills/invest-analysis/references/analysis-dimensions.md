# 分析维度参考

以下维度不是必须全部覆盖的清单，而是根据 SOUL.md 和当前语境**选择性深入**的方向。

## 定性维度

### 商业模式与护城河
- 公司靠什么赚钱？收入结构是否健康？
- 护城河是什么类型（品牌、网络效应、转换成本、规模经济、牌照/专利）？
- 护城河是在加深还是被侵蚀？证据是什么？

### 竞争格局
- 市场集中度如何？公司的市场地位变化趋势？
- 竞品的动态对公司构成什么威胁或机会？
- 行业是存量博弈还是增量市场？

### 管理层评估
- 管理层的资本配置能力如何？（看回购、分红、并购记录）
- 管理层说的和做的是否一致？（对比历史电话会承诺与实际执行）
- 有无治理风险？（股权结构、关联交易、管理层变动）

### 催化剂与时间节点
- 近期有什么可能改变公司价值的事件？
- 市场预期和实际情况的差距在哪里？

## 定量维度

### 盈利能力
- 毛利率、净利率趋势（至少看 3 年）
- ROE/ROIC 水平及变化方向
- 自由现金流与净利润的关系（现金利润比）

### 成长性
- 营收增速、利润增速趋势
- 增长是内生的还是靠并购？
- 增长的可持续性判断

### 财务健康
- 资产负债结构（负债率、有息负债比例）
- 现金流状况（经营性现金流是否覆盖资本开支）
- 应收账款/存货周转变化

### 估值
- 当前估值水平（PE、PB、EV/EBITDA 等，选适合该行业的指标）
- 与历史估值区间对比
- 与同业可比公司对比
- 如果 SOUL.md 中有明确的估值标准（如 DCF、安全边际要求），按其标准计算
